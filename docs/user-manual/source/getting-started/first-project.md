# Your First Project

Follow this step-by-step guide to create your first beat on the PGB-1.

## Starting Fresh

1. Press {kbd}`Menu`
2. Navigate to **Projects** and press {kbd}`A`
3. Select **New** and press {kbd}`A`
4. Confirm to create a new empty project

## Programming the Kick Drum

1. Press {kbd}`Track` to enter Track Mode
2. Press {kbd}`1` to select the Kick track
3. Press {kbd}`Edit` to enable pattern edit mode
4. Press these buttons to place kick notes: {kbd}`1`, {kbd}`5`, {kbd}`9`, {kbd}`13`

You now have a basic four-on-the-floor kick pattern!

## Adding the Snare

1. Hold {kbd}`Track` and press {kbd}`2` to switch to the Snare track
2. Press {kbd}`5` and {kbd}`13` to place snare hits on beats 2 and 4

## Creating a Hi-Hat Pattern

1. Hold {kbd}`Track` and press {kbd}`3` to switch to the Hi-Hat track
2. Press {kbd}`3`, {kbd}`5`, {kbd}`7`, {kbd}`8`, {kbd}`11`, {kbd}`15`, {kbd}`16` to create an interesting hi-hat rhythm

## Adding Variation with Step Settings

Let's make the hi-hats more dynamic:

1. Press {kbd}`Step` to enter Step Mode
2. Press {kbd}`1` to select step 1
3. Use {kbd}`←` to navigate to the **Condition** setting
4. Press {kbd}`↑` multiple times to set condition to **75%**
5. Press {kbd}`→` three times to reach the **Velocity** setting
6. Use the touch strip to lower the velocity to about one third

## Copying Steps

Copy the configured step to create consistent variation:

1. Hold {kbd}`Cpy/FX`
2. Press {kbd}`Step` to enter step copy mode
3. Press {kbd}`1` to select the source step
4. Press {kbd}`2`, {kbd}`6`, {kbd}`9`, {kbd}`10`, {kbd}`13`, {kbd}`14` to paste to those steps

## Adding Shuffle

1. Press {kbd}`Track` to return to Track Mode
2. Press {kbd}`→` multiple times until you reach the **Shuffle** page
3. Press {kbd}`↑` to increase the shuffle amount to taste

## Play Your Beat!

Press {kbd}`Play` to hear your creation!

## Quick Reference

| Action | Buttons |
|--------|---------|
| Select track | Hold {kbd}`Track` + {kbd}`1`-{kbd}`16` |
| Toggle step | {kbd}`Edit` then {kbd}`1`-{kbd}`16` |
| Copy step | Hold {kbd}`Cpy/FX` + {kbd}`Step` + source + destination |
| Play/Stop | {kbd}`Play` |
| Adjust volume | Hold {kbd}`Play` + {kbd}`↑` / {kbd}`↓` |
| Adjust BPM | Hold {kbd}`Play` + {kbd}`←` / {kbd}`→` |
