## Front Panel

### Power

- **Power Switch** (side) - Slide left to turn on, right to turn off
- **Power On LED** (top left, cyan) - Lit when the device is powered on
- **Battery Charge LED** (top left, red) - Lit when the battery is charging

### Navigation Buttons

- {kbd}`←` {kbd}`→` - Navigate between settings pages
- {kbd}`↑` {kbd}`↓` - Change parameter values
- {kbd}`A` - Confirm / Select / Enable
- {kbd}`B` - Back / Cancel / Disable

### Mode Buttons

- {kbd}`Track` - Enter [Track Mode](../modes/track-mode.md) (configure instruments and sounds)
- {kbd}`Step` - Enter [Step Mode](../modes/step-mode.md) (configure individual step parameters)
- {kbd}`Pattern` - Enter [Pattern Mode](../modes/pattern-mode.md) (configure pattern length and linking)
- {kbd}`Song` - Enter [Song Mode](../modes/song-mode.md) (arrange parts and chord progressions)

### Keyboard Section

The 16 numbered buttons ({kbd}`1` to {kbd}`16`) serve multiple purposes depending on
the current mode (see [Main Modes](../modes/index.md)).

### Control Buttons

- {kbd}`Menu` - Open the main menu
- {kbd}`Play` - Start/Stop the sequencer (also used for shortcuts when held)
- {kbd}`Edit` - Toggle pattern edit mode / Hold for chromatic keyboard
- {kbd}`Cpy/FX` - Hold for Live FX access and copy/paste operations

### Touch Strip

The capacitive touch strip at the bottom of the device allows for:

- Quick parameter value adjustment
- Live performance expression

## Connectors

From right to left:

- **Headphone/headset** (3.5mm stereo jack) - Connect headphone/headset or external speakers
- **Line Input** (3.5mm stereo jack) - Connect external audio sources
- **USB** (type-C) - For charging and firmware updates
- **MIDI Output** (3.5mm stereo jack, TRS type A) - Control external gear via MIDI protocol
- **MIDI Input** (3.5mm stereo jack, TRS type A) - Control the PGB-1 with
  external gear (MIDI keyboard or sequencer, Digital Audio Workstation (DAW), etc.)

## Internal Components

- **Built-in Speaker** - For monitoring without headphones
- **Internal Microphone** - For recording samples
- **Rechargeable Battery** - Provides portable operation

## Recovery

- **Reset Button** (small hole) - Use a paperclip to reset the device
- **Boot Button** (small hole) - For forced firmware update procedure
