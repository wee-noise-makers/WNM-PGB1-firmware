# Shortcuts

Quick reference for all PGB-1 keyboard shortcuts.

## Volume and BPM

| Action | Shortcut |
|--------|----------|
| Increase Volume | Hold {kbd}`Play` + Press {kbd}`↑` |
| Decrease Volume | Hold {kbd}`Play` + Press {kbd}`↓` |
| Increase BPM | Hold {kbd}`Play` + Press {kbd}`→` |
| Decrease BPM | Hold {kbd}`Play` + Press {kbd}`←` |

## Mute/Solo

| Action | Shortcut |
|--------|----------|
| Mute/Unmute Track | Hold {kbd}`Play` + Press {kbd}`1`-{kbd}`16` |
| Solo Track | Hold {kbd}`Play` + Press {kbd}`Track`, then {kbd}`1`-{kbd}`16` |

## Mode Selection

| Action | Shortcut |
|--------|----------|
| Enter Track Mode | Press {kbd}`Track` |
| Enter Step Mode | Press {kbd}`Step` |
| Enter Pattern Mode | Press {kbd}`Pattern` |
| Enter Song Mode | Press {kbd}`Song` |
| Enter Sample Mode | Hold {kbd}`Cpy/FX` + Press {kbd}`Edit` |

## Quick Selection (Without Mode Change)

| Action | Shortcut |
|--------|----------|
| Select Track | Hold {kbd}`Track` + Press {kbd}`1`-{kbd}`16` |
| Select Step | Hold {kbd}`Step` + Press {kbd}`1`-{kbd}`16` |
| Select Pattern | Hold {kbd}`Pattern` + Press {kbd}`1`-{kbd}`16` |
| Select Song Part | Hold {kbd}`Song` + Press {kbd}`1`-{kbd}`12` |

## Pattern Editing

| Action | Shortcut |
|--------|----------|
| Toggle Edit Mode | Press {kbd}`Edit` |
| Toggle Step On/Off | (In Edit Mode) Press {kbd}`1`-{kbd}`16` |
| Chromatic Keyboard | Hold {kbd}`Edit` + Press keys |
| Octave Down (Keyboard) | Hold {kbd}`Edit` + Press {kbd}`1` |
| Center Octave (Keyboard) | Hold {kbd}`Edit` + Press {kbd}`4` |
| Octave Up (Keyboard) | Hold {kbd}`Edit` + Press {kbd}`8` |

## Live FX

| Action | Shortcut |
|--------|----------|
| Toggle Live FX | Hold {kbd}`Cpy/FX` + Press {kbd}`1`-{kbd}`16` |

### Sequencing FX (Left Side)

| Button | Effect |
|--------|--------|
| {kbd}`1` | Roll 16th |
| {kbd}`2` | Roll 8th |
| {kbd}`3` | Roll Quarter |
| {kbd}`4` | Roll Beat |
| {kbd}`9` | Fill Steps |
| {kbd}`10` | Auto-fill Low |
| {kbd}`11` | Auto-fill High |
| {kbd}`12` | Auto-fill Build-up |

### Mixing FX (Right Side)

| Button | Effect |
|--------|--------|
| {kbd}`5` | Low Pass Filter |
| {kbd}`6` | Band Pass Filter |
| {kbd}`7` | High Pass Filter |
| {kbd}`8` | Stutter 1 |
| {kbd}`13` | LP Filter Sweep |
| {kbd}`14` | BP Filter Sweep |
| {kbd}`15` | HP Filter Sweep |
| {kbd}`16` | Stutter 2 |

## Copy/Paste

| Action | Shortcut |
|--------|----------|
| Copy Song Part | Hold {kbd}`Cpy/FX` + {kbd}`Song` + source + dest |
| Copy Track | Hold {kbd}`Cpy/FX` + {kbd}`Track` + source + dest |
| Copy Pattern | Hold {kbd}`Cpy/FX` + {kbd}`Pattern` + source + dest |
| Copy Step | Hold {kbd}`Cpy/FX` + {kbd}`Step` + source + dest |

## Quick Parameter Control

| Action | Shortcut |
|--------|----------|
| Live Parameter Adjustment | Hold {kbd}`Cpy/FX` + Use Touch Strip |
