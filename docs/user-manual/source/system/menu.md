# Main Menu

The main menu provides access to system functions, project management, and device settings.

## Accessing the Menu

Press {kbd}`Menu` to open the main menu.

Navigate with {kbd}`←` / {kbd}`→` and select with {kbd}`A`.

## Menu Items

### Projects

Save, load, create, delete [projects](../deep-dive/projects.md)

```{figure} ../../../assets/OLED-screenshots/framed/PGB1-OLED-menu-projects-framed.png
:alt: Projects Menu
```

### Project Mixer

Per project settings of gains and audio levels for tracks and track effects.
See [Mixing](../deep-dive/mixing.md).

*Available from firmware 1.2.0*

```{figure} ../../../assets/OLED-screenshots/framed/PGB1-OLED-menu-project-mixer-framed.png
:alt: Project Mixer Menu
```

### Custom Waveform

Draw and edit custom waveforms. See [Custom
Waveforms](../deep-dive/custom-waveforms.md).

*Available from firmware 1.2.0*

```{figure} ../../../assets/OLED-screenshots/framed/PGB1-OLED-menu-custom-waveform-framed.png
:alt: Custom Waveform Menu

Customize live sequencing and mixing effects. See [Live

### Update Mode

Enter firmware update mode. See [Firmware Updates](firmware.md).

:::{warning}
Entering update mode will save the current project and reboot the device.
:::

```
