# Main Menu

The main menu provides access to system functions, project management, and device settings.

## Accessing the Menu

Press {kbd}`Menu` to open the main menu.

Navigate with {kbd}`←` / {kbd}`→` and select with {kbd}`A`.

## Menu Items

### Projects

Save, load, create, delete [projects](../deep-dive/projects.md)

### Custom Waveform

Draw and edit custom waveforms. See [Custom
Waveforms](../deep-dive/custom-waveforms.md).

*Available from firmware 1.2.0*

### Inputs

Configure [audio inputs and FX](../audio-midi/input.md).

### MIDI Settings

Configure [MIDI input and output](../audio-midi/midi.md).

### Update Mode

Enter firmware update mode. See [Firmware Updates](firmware.md).

:::{warning}
Entering update mode will save the current project and reboot the device.
:::

### System Info

View device information:

- Firmware version
- Other system details

Useful for:

- Checking if you need to update
- Support requests
- Troubleshooting
